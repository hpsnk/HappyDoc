## npm 镜像地址

```
npm get registry

npm config set registry http://registry.npmmirror.com/

npm config set registry https://registry.npmjs.org/
```

## 安装vue脚手架(vue cli)

### vue 2.x 脚手架
```
npm install vue-cli -g
```

### vue 3.x 脚手架
```
npm install @vue/cli -g
```

### 卸载
```
npm uninstall -g @vue/cli@版本号
npm uninstall -g @vue/cli
```

### 确认版本号
```
vue -V
```
