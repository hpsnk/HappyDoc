
* 重置pi用户的密码
    ```
    sudo passwd pi
    ```

* 登录pi用户 
    ```
    su pi
    ```

## root用户

* 重置root用户的密码
    ```
    sudo passwd root
    ```

* 登录root用户
    ```
    su root
    ```

* 解锁root
    ```
    sudo passwd --unlock root
    ```

* 锁定root
    ```
    sudo passwd --lock root
    ```
